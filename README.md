# phonebook_in_C

Algorithm "phonebook_in_C.c"

Makefile commands

	make: compiles code
	make clean: deletes files
	make run: executes prog without args

The programm needs input file (txt), with contacts, example:

	Dimitris_Greasidis 6978399094

Example of run:

	./phonebook test

Compare files

	diff -b myout1.txt stdout1.txt
